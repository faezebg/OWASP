# OWASP Crypto Lab

این پروژه شامل اسکریپت‌های آموزشی و آزمایشی برای تمرین مفاهیم رمزنگاری و تست نفوذ مربوط به مدیریت رمز عبور است.

## محتویات
- اسکریپت‌های اصلی: `app.py`, `app-safe.py`, `cracker.py`
- فهرست پسوردها: `passwords.txt`, `passwords-safe.txt`, `passProbablyList.txt`
- قالب‌های HTML در پوشه `templates/`

## نصب و اجرا (محلی)
1. مطمئن شوید Python نصب است.
2. (اختیاری) یک virtualenv بسازید و فعال کنید.
3. اجرای برنامه‌ها با `python app.py` یا `python app-safe.py` بر حسب نیاز.
